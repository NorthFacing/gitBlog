# 特殊字符

### 文件树结构符号
```
conf
├── ca.key  -- caroot 的私钥
├── ca.csr  -- caroot 的请求文件
├── ca.pem  -- caroot 的根证书，也可以以 .crt 结尾
├── client
│   ├── client.csr  -- client 请求签名文件
│   ├── client.key  -- client 的私钥
│   └── client.pem  -- client 的证书（由 caroot 签发），也可以以 .crt 结尾
└── server
    ├── server.csr
    ├── server.key
    └── server.pem
```
