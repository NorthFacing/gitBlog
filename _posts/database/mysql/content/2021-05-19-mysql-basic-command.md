## 事务

```mysql
SELECT @@autocommit;
show variables like 'autocommit';
```

### 手动管理事务
```mysql
begin; # begin 回车后并不会立即开启事务，而是在执行真正的CRUD语句后才开启事务
commit; # 提交事务，操作生效并持久化
rollback; # 回滚，撤销所有操作（可能出现脏读）
```
